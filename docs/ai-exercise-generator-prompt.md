**Objective**: Transform number 1 → target using 4 operations in minimum moves.

- `reverse(n)`: Flip digits (123 → 321)
- `sum_digits(n)`: Add digits (128 → 11) 
- `append1(n)`: Add '1' to end (12 → 121)  
- `double(n)`: Multiply by 2 (64 → 128)

**Constraints**: Valid numbers 1-10,000, must be solvable from 1.
